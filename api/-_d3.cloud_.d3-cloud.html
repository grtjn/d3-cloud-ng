d3-cloud source

Description

Angular directive wrapping the d3-cloud library.

Attributes

NameTypeDescription
events Object

Optional. An object with a property for each event callback function to be supported. Default: {}.
filter Function

Optional. A function that filters words. Invoked for each word, should return true to retain the word, false to skip. Defaults to comparing against ignoreList for legacy purposes.
font String

Optional. The name of the font to use. Default: Impact.
ignoreList Array

Deprecated. Optional. An array of word names to ignore. Default: [].
padding Integer

Optional. The padding to apply between words. Default: 5.
rotate Function

Optional. A function reference that calculates rotation per word. Takes word object, and index in 'words' array. Default: alternating 45 degree left/right.
slope-base Integer

Optional. The minimum size for words. Default: 2.
slope-factor Integer

Optional. The scale factor applied to scores. Default: 30.
words Array

A binding to an array of objects with name, score and optional color properties.

Example

<d3-cloud events="ctrl.wordEvents" font="Impact" filter="ctrl.filter" padding="5"
    rotate="ctrl.rotateWord" slope-base="2" slope-factor="30" words="ctrl.words">
  </d3-cloud>
